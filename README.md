# AI Resume Frontend (Vite + React)

## Local

- Install: `npm install`
- Dev: `npm run dev`
- Build: `npm run build`

## Deploy (Vercel)

This repo includes `vercel.json` configured for Vite.

Recommended Vercel environment variable:

- `VITE_API_BASE_URL` = your deployed backend URL (e.g. Render service URL)
